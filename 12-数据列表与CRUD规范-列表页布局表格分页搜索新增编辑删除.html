📐 一、列表页整体布局框架

标准数据列表页由 5 个区域组成：页面标题+操作区 → 搜索条件区 → 工具栏 → 数据表格 → 分页器

区域位置内容规范要点
① 页面标题区顶部标题 + 操作按钮（新增/导出等）主操作按钮在右侧，用 primary 样式
② 搜索条件区标题下方N 个搜索字段 + 搜索/重置按钮 + 已选条件标签字段关联数据字段类型，输入即搜索（300ms防抖）
③ 工具栏表格上方批量操作按钮 + 选中计数 + 视图切换选中项 ≥1 时才显示批量操作
④ 数据表格核心区域表头（可拖拽列宽+点击排序） + 数据行 + 行内操作行高 ≥48px，列宽可拖拽，hover 高亮
⑤ 分页器表格下方总数 + 每页条数 + 页码 + 跳转默认 20 条/页，选项 10/20/50/100

🔗 二、搜索条件 ↔ 数据字段映射规范

搜索条件应根据数据字段类型选择对应的搜索控件，而非全部用文本输入框。

数据字段类型搜索控件匹配方式示例说明
短文本（姓名/编号）文本输入框模糊 LIKE %keyword%用户名、订单号、SKU防抖 300ms，支持 Enter 触发
手机/电话文本输入框精确 = 或 右模糊138****1234自动过滤非数字字符
状态/类型（有限枚举）下拉选择精确 =启用/禁用/待审核必须提供「全部」选项
日期/时间日期范围选择器范围 BETWEEN2024-01-01 ~ 2024-06-30提供快捷选项
金额/数量数值范围（min~max）范围 BETWEEN金额 100 ~ 1000注意精度，用 DECIMAL
布尔/开关下拉（是/否/全部）精确 =是否已审核可用 Switch 代替
长文本/富文本不提供搜索—文章正文、备注性能考虑

📊 三、数据表格列规范

列宽策略、文字截断、排序标识、操作列固定——数据表格四大核心规范。

3.1 列宽分配策略

数据类型推荐宽度最小宽度截断策略示例
复选框列48px48px不可拖拽☑️
序号/ID80~100px60px不截断10001
短文本120~160px100px… 省略张三
中文本200~300px150px… 省略 + title关于2024年度…

3.2 表头排序规范

状态图标表头样式说明
默认（未排序）⇅（灰色）悬停才显示图标hover 时图标出现
升序（ASC）▲（青色）th 加 sorted-asc 类图标始终显示
降序（DESC）▼（青色）th 加 sorted-desc 类再次点击切换方向
💡 排序交互：点击列头 → 升序 → 再点 → 降序 → 再点 → 取消排序（三态循环）| 同时只允许单列排序

规范项规范值说明
拖拽手柄区域表头 th 右侧 6px 宽cursor: col-resize
拖拽指示线2px 宽 青色竖线position: absolute
最小列宽60px拖拽时检测 min-width

🔘 五、操作按钮规范

操作按钮分为页面级操作、行内操作、批量操作三类。

类型位置按钮样式示例操作
页面级操作页面标题右侧primary 实心按钮新增、导入、批量导出
行内操作（主要）每行操作列ghost 文字按钮查看、编辑
行内操作（危险）每行操作列danger-ghost 红色文字删除
批量操作工具栏左侧outline 边框按钮批量删除、批量启用
更多操作（≥4个）操作列最后⋯ 下拉菜单复制、分享、归档

✏️ 六、新增 / 编辑 / 删除 / 详情（CRUD 完整流程）

增删改查四大操作的页面形态、交互流程和设计规范。

新增/编辑 —— 共用同一表单

规范项规范值说明
展示方式弹窗（Modal）或 独立页面字段 ≤10 用弹窗
表单布局双列栅格（1fr 1fr）长字段跨两列
必填标识红色 * 号在标签后非必填标「（选填）」
校验时机失焦（onBlur）+ 提交前双重保险
提交成功关闭弹窗 + 刷新列表 + Toast「保存成功」

删除 —— 二次确认弹窗

规范项规范值说明
触发方式行内「删除」按钮 或 批量删除红色文字/按钮
确认弹窗居中 Modal 420px小尺寸弹窗
文案「确定删除 XXX 吗？此操作不可撤销。」说出具体影响
按钮「取消」（ghost）+ 「确定删除」（danger）危险按钮用红色

CRUD 操作状态流转

➕ 新增
① 点击「新增」→ 打开表单（空）
② 填写 → 失焦校验 → 提交
③ 后端返回成功 → 关闭弹窗
④ Toast「新增成功」→ 跳到第一页
✏️ 编辑
① 点击「编辑」→ 打开表单（回填）
② 修改 → 失焦校验 → 提交
③ 成功 → 关闭弹窗 + Toast
④ 当前页刷新（刷新该行）
🗑 删除
① 点击「删除」→ 确认弹窗
② 确认 → 提交删除请求
③ 成功 → 关闭弹窗 + Toast
④ 列表刷新（空页则回退）
💡 关键原则：① 新增成功后跳到第1页；② 编辑/删除成功后停留当前页；③ 删除导致当前页为空时自动回退；④ 操作完成后 Toast 反馈持续 3 秒。

✅ 八、CRUD规范检查清单

  • 列表页包含 5 个标准区域（标题、搜索、工具栏、表格、分页）
  • 搜索条件根据数据字段类型选择对应控件
  • 表格列宽合理分配，支持拖拽调整
  • 表头支持点击排序，三态循环
  • 操作按钮分类清晰（页面级/行内/批量）
  • 新增/编辑使用弹窗，字段≤10用双列栅格
  • 删除操作有二次确认弹窗
  • 分页默认 20 条/页，显示总数和跳页
  • 所有操作完成后给出 Toast 反馈
