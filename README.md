# 📐 设计系统规范库 · Design System Specs

面向中后台系统的一站式设计开发规范文档，覆盖从设计基础到安全认证的 **16 层全链路**。

浏览器打开 `index.html` 即可使用，左侧导航 + iframe 内嵌浏览，自带搜索与分组折叠。

---

## 项目结构

```
├── index.html                              # 总览导航（12 层分组 · 搜索 · 标签快捷跳转）
├── 01-设计基础规范-颜色字体排版表格反馈表单.html
├── 02-布局框架规范-栅格间距页面框架响应式层级.html
├── 03-组件交互规范-按钮输入框弹窗选择标签页开关.html
├── 04-图标规范-Icon库尺寸风格使用场景.html
├── 05-数据可视化规范-图表配色卡片数据统计面板.html
├── 06-空状态异常规范-空数据404错误加载失败.html
├── 07-导航菜单规范-面包屑侧边菜单分页步骤条.html
├── 08-搜索筛选规范-搜索框筛选面板标签过滤排序.html
├── 09-数据库设计规范-字段设计命名索引表结构.html
├── 10-控件规范库-日期选择下拉搜索穿梭框文件上传树形选择.html
├── 11-字段类型映射规范-数据库字段与表单控件对应关系.html
├── 12-数据列表与CRUD规范-列表页布局表格分页搜索新增编辑删除.html
├── 13-编程语言规范-PHP-Python-CSharp-Java.html
├── 14-数据库操作规范-DBHelper与ORM设计.html
├── 15-Nginx配置规范-反向代理负载均衡HTTPS配置生成器.html
├── 16-前端性能优化规范-CSS引入JS位置DOM优化重排重绘.html
├── 17-文本内容展示规范-文章详情标题作者日期段落配图.html
├── 18-数据交互规范-API请求响应格式与错误处理.html
├── 19-身份认证与安全规范-登录注册权限JWT安全策略.html
├── README.md
└── LICENSE
```

## 使用方式

直接用浏览器打开 `index.html`，通过左侧导航切换各规范；或单独打开任一 HTML 文件查看对应规范。无需构建工具、无需后端服务。

## 16 层规范架构

| 分层 | 主题色 | 包含文档 |
|:---|:---:|:---|
| 设计基础层 | `#0066CC` | 01-设计基础：颜色 / 字体 / 表格 / 反馈 / 表单 |
| 布局框架层 | `#8B5CF6` | 02-布局框架：栅格 / 间距 / 页面框架 / 响应式 / z-index |
| 组件交互层 | `#10B981` | 03-组件交互：按钮 / 输入框 / 弹窗 / 选择器 / 标签页 / Tooltip |
| 视觉资产层 | `#F59E0B` | 04-图标规范：Icon 库 / 尺寸梯度 / 颜色语义 / 使用场景 |
| 数据展示层 | `#EF4444` | 05-数据可视化 + 06-空状态异常：图表配色 / 卡片 / 骨架屏 / 404 |
| 数据管理层 | `#0EA5E9` | 12-数据列表与 CRUD：列表布局 / 表格分页 / 搜索联动 / 增删改查 |
| 导航操作层 | `#EC4899` | 07-导航菜单 + 08-搜索筛选：面包屑 / 侧边菜单 / 筛选面板 / 排序 |
| 数据持久层 | `#06B6D4` | 09-数据库设计 + 11-字段类型映射：命名 / 索引 / 表结构 / 控件映射 |
| 复合控件层 | `#84CC16` | 10-控件规范库：日期选择 / 下拉搜索 / 穿梭框 / 文件上传 / 级联 |
| 编程语言层 | `#6366F1` | 13-多语言编码规范：PHP · Python · C# · Java |
